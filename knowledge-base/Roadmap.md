# Roadmap

Bu not, eski plan dokumanlarindan knowledge base'e tasinan backlog'un kisa haritasidir.

## Sprint 2 Temalari

- Repeat mode
- i18n
- Lyrics
- Config file
- Search filters
- Terminal title
- Add to queue
- Search history
- Search screen back navigation
- Search play sonrasi auto-unpause davranisi

## Sprint 3 Temalari

- Volume control
- Coklu dil destegi

## Dikkat Ceken Plan Notlari

### Ses Kontrolu

Sprint 3 planinda `+` / `-` ile ses kontrolu tanimlanmis. Kodda bu davranis uygulanmis durumda ve player ekraninda ses seviyesi gosteriliyor.

### i18n

Plan tarafinda hala acik gorunuyor. Bunun ana etkisi:

- UI metinlerini merkezilestirmek
- `tr` ve `en` locale dosyalari eklemek
- Sistem dili veya config tabanli dil secimi sunmak
